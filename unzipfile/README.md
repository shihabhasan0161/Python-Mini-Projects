## This does as the name suggests - it unzips a zip file.
## Also, it zips a folder back into a zip file.

### Why I wrote this script?
I just wanted a simple script to automate the unzipping and zipping of files/folders. Because I often had to extract zip files manually.

### How to use:
copy unzip.py to your working directory.
then run the script from terminal as follows:
To unzip a file:
```bash
python unzip.py filename.zip
```

To zip a folder:
```bash
python zip.py foldername
```

You may copy paste your file path instead of filename.zip or foldername.
